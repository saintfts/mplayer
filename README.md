# MPlayer
A really basic music player built in Flutter and just_audio

<img width="943" height="1023" alt="image" src="https://github.com/user-attachments/assets/888e0f55-30f8-4ffe-9fef-37ea652a8479" />

### Current version features:

* Playback functionality
* Music rewinding
* Volume slider
* Settings page for setting tracks path

### Plans:
* Image covers for each song
* Albums support
* More customization
* Music streaming services playback
* Music ordering & shuffling & shuffling + ordered playback
* Background playback
* Adjust UI for specific platforms (Mobile/Desktop)
* Proper code refactoring & performance optimizations
* etc.
